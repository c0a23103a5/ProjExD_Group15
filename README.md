# GROUP15/

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* ロックマンlike

## ゲームの遊び方
* 矢印キーで主人公を操作し，ボスキャラクターを倒したらゲームクリア

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画
* 矢印キーで主人公を動かす
* スペースキーでビーム発射
* 撃ち落とすとスコア加算
* 敵機が爆弾を発射
* （第四回演習のコード）

### 分担追加機能
* 重力
* チャージショット(C0B23157担当)
* 体力(主人公は体力表示あり)(C0B23065担当)
* ボスキャラ
* ステージ
